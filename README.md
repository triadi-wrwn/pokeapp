This is a project build with:
- [Nextjs 14](https://nextui.org/)
- [shadcn/ui](https://ui.shadcn.com)
- NodeJs >18
- [pnpm](https://pnpm.io/)

## Getting Started

Run the development server:

```bash
pnpm install
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
